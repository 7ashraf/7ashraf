## About Me

I'm a **Software Engineer** focused on building modern web products with **TypeScript, Next.js, Vercel, Supabase, and serverless architecture**.

I work as a software engineer, freelance on real-world products, build open-source SaaS projects, and contribute to open-source ecosystems including projects around **Vercel** and **Supabase**. I also have backend depth across **Node.js** and **ASP.NET Core** when products need more traditional API, service, or enterprise integrations.

I enjoy both sides of software:  
- building polished products that users can actually use  
- going deep into systems, infrastructure, architecture, and how things work under the hood  

Currently, I'm learning more about **deep infrastructure**, **scalable systems**, and **system design**.
